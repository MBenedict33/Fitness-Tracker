# Fitness Tracker

Personal fitness tracker with Supabase sync.

## Deploy to Vercel

1. Push this folder to a GitHub repo
2. Go to vercel.com → New Project → Import that repo
3. Add these Environment Variables in Vercel:
   - `REACT_APP_SUPA_URL` = https://jdoyqjhqlusfosfqlxti.supabase.co
   - `REACT_APP_SUPA_KEY` = (your anon key)
4. Click Deploy

## Local development

```bash
npm install
npm start
```
